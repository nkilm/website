# Umami Cloud

Umami Cloud is a hosted version of the Umami application run by the creators of Umami.

## Signup

You can sign up for free at [https://cloud.umami.is/signup](https://cloud.umami.is/signup).

Details on each plan can be found on the [Pricing](https://umami.is/pricing) page.

## Benefits

- **Fully managed**: Get all the latest updates without having to self-host and manage your own servers.
- **Scalable**: Umami provides seamless scalability, allowing you to handle increasing data volumes and user demands without compromising performance.
- **Cost-efficient**: Pay only for what you use with our simple pricing model.
- **Compliant**: Servers are hosted in the EU to comply with GDPR regulations.
